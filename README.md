Twitter Sentiment Analysis using RoBERTa

This project performs Twitter Sentiment Analysis using a pretrained RoBERTa model fine-tuned to classify tweets into Positive or Negative sentiments.

The workflow includes data preprocessing, model fine-tuning, accuracy evaluation, and exporting the trained model to ONNX for efficient deployment.
